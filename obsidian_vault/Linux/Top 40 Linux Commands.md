### Navigation Commands

| No. | Command | Description                |
| --- | ------- | -------------------------- |
| 1   | `cd`    | Change directory           |
| 2   | `pwd`   | Print working directory    |
| 3   | `ls`    | List directory contents    |
| 4   | `cd ..` | Change to parent directory |
| 5   | `cd ~`  | Go to home directory       |
